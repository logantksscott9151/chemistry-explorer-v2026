## Overview

Chemistry Explorer is a web-based study tool that helps users examine chemistry concepts through interactive element data and trend charts. It lays out the periodic table in a format that supports side-by-side comparison of properties, detailed element inspection, and quick recognition of patterns across the table, all within the same page.

This project is intended for students, teachers, and anyone who wants a fast way to browse essential chemistry ideas in a browser. Built with Chart.js visualizations and vanilla JavaScript, it stays lightweight while still providing a practical set of graphical references for element behavior.

## What it includes

- Interactive periodic table explorer for navigating elements
- Ionization energy charts for comparing element behavior
- Electron affinity comparison views
- Melting point and boiling point visualizations
- Electronegativity trend analysis across the table
- Atomic radius trend charts for property comparison
- Electron configuration display for individual elements
- Oxidation states reference and element classification

## How to use it

Launch the app in your browser and pick an element from the periodic table to inspect its properties and classification.

The included charts make it easy to compare trends like ionization energy, electron affinity, electronegativity, and atomic radius. The layout is suited to quick revision, classroom use, and self-guided review of how elements relate to one another.

## Configuration

Chemistry Explorer is delivered as a browser-based HTML project, so most configuration lives in the source files rather than a standalone installer.

To modify labels, data sources, chart behavior, or layout details, edit the HTML, CSS, and JavaScript files in the repository. Chart rendering is handled with Chart.js, while the interactive logic is written in vanilla JavaScript.
